# 投稿テンプレート v3

## 共通制約（4型共通）

- **10枚基準の二山構造**（挑発1 + 自分事化1 + 結論予告1 + 既知解説2 + REVEAL1 + 未知解説2 + まとめ表1 + CTA1）
- ±1枚まで可変。ただしREVEALスライドは省略不可
- 文字数固定なし。伝わる最小限。基本3行以内（削れないなら4行まで）
- 1スライド1メッセージ
- 各スライドにスタイルメタデータ（テキスト・サイズ・色）を付与 — **必ずテキスト形式で出力。テーブル形式NG**
- まとめ表は必ずMarkdown表形式で出力。1行あたり3〜4列
- **全スライドにBB的な一言（ツッコミ/怒り/問いかけ/損失指摘）を入れる**
- **論文著者名・学術略語・専門用語の無補足使用はスライド本文で一切禁止**（SKILL.md Part 2参照）
- **1スライドのレイヤー上限は3つ。** 4つ以上になる場合は2枚に分割する（SKILL.md Part 3参照）
- **数値には必ず比較対象をセットにする**（通常値/倍率/身近な例え。SKILL.md Part 3参照）
- **スライドは結論と数字だけ。** 根拠詳細・逸話列挙・Q&Aはキャプションに回す

## フック3点セット（★1〜3枚目とタイトル必須）

| 要素 | 運用 |
|---|---|
| 方向性明示 | タイトル冒頭と1枚目で「何を学べるか」の輪郭を宣言 |
| 謎フレーム | 答えの半分を `「？？？」` で隠し、6枚目REVEALで正体を公開 |
| 損失フレーム | 「大損」「損してます」「半分も効いてない」で行動変容を迫る |

## スタイル定義（全カテゴリ共通）

| 色 | 用途 |
|---|---|
| 黒 | 基本テキスト |
| 赤 | 最重要の一語・REVEAL（スライド内1〜2箇所まで） |
| 青 | 数値・単位（本文より一段階大きく） |

## CTAの選び方（全カテゴリ共通）

- 型1. 数式型 `A(単位) × B(単位) = ?` — 数値勝負テーマ向け
- 型2. 共感型 `好きな〇〇コメントしてくれ！！／俺は〇〇` — 人物・推し・選択テーマ向け

詳細は SKILL.md Part 5 参照。

## ingredient（成分解説系）

**コンセプト: 「〇〇を単体で摂るのは大損、鍵は〇〇×『？？？』」**

### タイトル

**型A. クイズ型:**
1. `{成分}、{現状のやり方}は大損！{既知要素}×「？？？」が鍵！`
2. `{成分}の吸収/効果を倍にする「？？？」、答えられる？`

**型B. 不安煽り＋言い切り型:**
3. `{現状のやり方}してる君、「？？？」足してないから{損失量}`
4. `{既知要素}だけで{成分}飲んでるやつ、「？？？」知らないと一生変わらない`

### 10枚構成

```
スライド 1（挑発）       ← {成分}、{現状のやり方}は大損！{既知要素}×「？？？」が鍵！
スライド 2（自分事化）   ← {現状のやり方}で{成分}飲んでませんか？…それ、{損失表現}
スライド 3（結論予告）   ← 結論！「{既知要素}」と「？？？」を一緒に摂れ！答えは後ほど→
スライド 4〜5（既知解説）← 成分Aの仕組み（比喩で語る。論文名ゼロ）
スライド 6（REVEAL）★  ← 見落としがちな「？？？」の正体＝{未知要素}
スライド 7〜8（未知解説）← 未知要素の仕組み（比喩で語る）+ 相乗効果
スライド 9（まとめ表）   ← 相乗効果の整理
スライド 10（CTA）
```

### 感情設計

- 挑発: ドキッ・損失警戒
- 自分事化: 「自分のこと？」
- 結論予告: 続きが気になる
- 既知解説: 「へぇ、そういう仕組みか」
- REVEAL: 「そういうことか！」★ピーク
- 未知解説: 「マジか！」
- まとめ表: 信頼・保存したい
- CTA: 参加欲求

スライド 4〜5（既知解説）← AとBの表面的な差（数字と比喩で語る）
スライド 6（REVEAL）★  ← 見落とされてる「？？？」の正体＝{新軸}
スライド 7〜8（未知解説）← 新軸で再評価した仕組み・相乗効果

## entertainment（エンタメ系）

**コンセプト: 「固有名詞の表の行動の裏にある『？？？』を真似しないと意味ない」**

### タイトル

**型A. クイズ型:**
1. `{人物}の{有名な行動}、真似するだけじゃ大損！鍵は「？？？」`
2. `{人物}が{有名な行動}してる本当の理由「？？？」、分かる？`

**型B. 不安煽り＋言い切り型:**
3. `{有名な行動}だけ真似してる君、「？？？」やってないから意味ない`
4. `{人物}の真似で{表面要素}増やしてるやつ、「？？？」無視したら逆効果`

### 10枚構成

```
スライド 1（挑発）       ← {人物}の{有名な行動}、表面だけ真似すると大損！鍵は「？？？」
スライド 2（自分事化）   ← {有名な行動}やってるだけの君…それ、半分も真似できてません
スライド 3（結論予告）   ← 結論！{人物}がやってたのは「{有名な行動}」と「？？？」 答えは後ほど→
スライド 4〜5（既知解説）← 表の逸話の核心（固有名詞の有名エピソード）
スライド 6（REVEAL）★  ← 実は裏でやってた「？？？」の正体＝{隠れた実践}
スライド 7〜8（未知解説）← 科学的解剖＋セルフツッコミで爆発
スライド 9（まとめ表）   ← 表の逸話 × 裏のポイント × 科学
スライド 10（CTA）
```

## debunk（誤認訂正系）

**コンセプト: 「古い常識を避けても意味ない、本当に注意すべき『？？？』を知らないと損し続ける」**

**型B. 不安煽り＋言い切り型:**
3. `「{誤認}」で逃げてるやつ、「？？？」を見落としてるから{損失}`
4. `{誤認}信じてる情弱、「？？？」知らないから一生その体型`

### 10枚構成

```
スライド 1（挑発）       ← {誤認}…それ、10年前の情弱知識！鍵は「？？？」
スライド 2（自分事化）   ← まだ{誤認に基づく行動}してませんか？…それ、{損失表現}
スライド 3（結論予告）   ← 結論！怖いのは{誤認対象}じゃない、「？？？」だ！答えは後ほど→
スライド 4〜5（既知解説）← 誤認の根拠（弱い論文）を証拠で叩く
スライド 6（REVEAL）★  ← 本当に注意すべき「？？？」の正体＝{本当の注意点}
スライド 7〜8（未知解説）← 正しい実践の仕組み＋実践ライン
スライド 9（まとめ表）   ← 誤認 vs 事実 vs 本当の注意点（3列）
スライド 10（CTA）
```
