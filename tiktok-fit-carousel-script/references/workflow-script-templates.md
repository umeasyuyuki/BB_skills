# 投稿テンプレート

## 共通制約

- 9〜12枚可変（挑発1 + 正解2〜3 + 証拠4〜6 + まとめ表1 + CTA1）
- 文字数固定なし。伝わる最小限。基本3行以内（削れないなら4行まで）
- 1スライド1メッセージ
- 各スライドに感情ラベルを付与
- 各スライドにスタイルメタデータ（テキスト・サイズ・色）を付与
- まとめ表は必ずMarkdown表形式 `| 列1 | 列2 |` で出力。1行あたり3〜4列
- 列名はテーマに合わせて柔軟に設定する（固定しない）

## スタイル定義（全カテゴリ共通）

| 色 | 用途 |
|---|---|
| 黒 | 基本テキスト |
| 赤 | 最重要の一語（スライド内1〜2箇所まで） |
| 青 | 数値・単位（本文より一段階大きく） |

## CTAの設計原則（全カテゴリ共通）

テーマから「視聴者が自分の数値・状況・経験を1つ書けば承認される問い」を毎回導出する。
命令形（「〜しろ」）は使わない。「教えてほしい」「聞かせてほしい」「コメントで教えて」を基本にする。

## compare（比較系）`〇〇 vs 〇〇`

**構成順:** 挑発 → 正解 → 証拠 → まとめ表 → CTA

```
スライド 1（挑発）    ← テーマで思想を込める。情弱インフルエンサー等のワード活用
スライド 2〜3（正解）  ← 比較の結論を先出し
スライド 4〜8（証拠）  ← 各指標のデータを並べる
スライド 9〜11（まとめ表） ← 比較表（このカテゴリで最も映える）
スライド 最終（CTA）
```

**感情設計:**
- 挑発: ドキッ
- 正解: 「え、そっちなの？」
- 証拠: データで納得
- まとめ表: 信頼・保存したい
- CTA: 参加欲求

**CTA例:**
「どっち使ってる？量も含めて教えてほしい（例: A製品 5g）」
「今のプロテイン、1食あたりいくら？コメントで教えて」

**まとめ表例:**

| 比較項目 | 〇〇 | 〇〇 | 判定 |
|---|---|---|---|
| ... | ... | ... | ... |

## ingredient（成分解説系）`牛乳 / アシュワガンダ等`

**構成順:** 挑発 → 正解 → 証拠 → まとめ表 → CTA

```
スライド 1（挑発）    ← 「〇〇の本当の効果、知ってる？」型
スライド 2〜4（正解）  ← この成分で何が起きるか（結論先出し）
スライド 5〜9（証拠）  ← メカニズム・論文データ
スライド 10（まとめ表） ← 成分・用量・タイミング・効果
スライド 最終（CTA）
```

**感情設計:**
- 挑発: ドキッ
- 正解: 「へえ、そういう仕組みか」
- 証拠: 「え、マジで？」
- まとめ表: 信頼・実践したい
- CTA: 参加欲求

**CTA例:**
「一日の〇〇摂取量、コメントで教えてほしい（例: 5g / 1錠 / 飲んでない）」
「摂り始めてから何か変化あった？コメントで聞かせてほしい」

**まとめ表例:**

| 対象 | 用量 | タイミング | 期待効果 |
|---|---|---|---|
| ... | ... | ... | ... |

## entertainment（エンタメ系）`ラランド西田 / 横川くん等`

**構成順:** 挑発 → 逸話 → 証拠 → まとめ表 → CTA

```
スライド 1（挑発）    ← 「〇〇（固有名詞）の食生活を科学した」型
スライド 2〜4（逸話）  ← 固有名詞を絡めた逸話の核心
スライド 5〜8（証拠）  ← フィットネス/ヘルスケアの科学で解釈
スライド 9（まとめ表） ← 逸話×科学の対応表
スライド 最終（CTA）
```

**口調:** 他カテゴリより柔らかくてよい。BBトーンは維持しつつ「面白い発見をした」ベース。キレ不要。

**感情設計:**
- 挑発: 「なんか面白そう」
- 逸話: 引き込まれる
- 証拠: 「ちゃんと科学的に話してる」
- まとめ表: 面白い・共有したい
- CTA: 参加欲求・遊び心

**CTA例:**
「〇〇に何を食わせたい？コメントで教えてほしい」
「こういう逸話、他に知ってたら教えてほしい」

**まとめ表例:**

| 逸話 | 科学的解釈 | 実際の効果 |
|---|---|---|
| ... | ... | ... |

## debunk（誤認訂正系）`プロテインは腎臓を悪くする等`

**構成順:** 挑発 → 証拠 → 正解 → まとめ表 → CTA

証拠を先に出して誤認を叩いてから、正しい認識を提示する。

```
スライド 1（挑発）    ← タイトル・テーマで思想を出す（「〇〇は情弱」型）
スライド 2〜6（証拠）  ← 論文・データで誤認を叩く
スライド 7〜9（正解）  ← 正しい認識・実践
スライド 10（まとめ表） ← 誤認 vs 事実の対照表
スライド 最終（CTA）
```

**口調:** このカテゴリだけキレた口調OK。

**感情設計:**
- 挑発: ドキッ・「自分のこと？」
- 証拠: 怒りの種・「え、マジで？」
- 正解: 「助かる」・腑に落ちる
- まとめ表: 信頼・整理された
- CTA: 参加欲求・共感

**CTA例:**
「これ信じてた期間、コメントで教えてほしい（例: 3年）恥ずかしくないよ」
「周りにまだ信じてる人いる？何人くらいか教えてほしい」

**まとめ表例:**

| 誤認 | 事実 | 根拠強度 |
|---|---|---|
| ... | ... | ... |
